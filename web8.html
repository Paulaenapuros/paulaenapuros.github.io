PUBLICACIÓN Y ACTUALIZACIÓN DE CONTENIDOS
Hosting o alojamiento web es el espacio disponible en un servidor de internet para el almacenamiento de las páginas web y el resto de archivos relacionados.
Las páginas web son archivos que ocupan un espacio físico en bytes.

FTP transmite archivos en internet que permite la comunicación entre el usuario y el servidor.

Gestor de contenidos(CMS) es una aplicación que permite editar y modificar los contenidos de páginas web dinámicas a partir de formatos predefinidos y de contenidos almacenados en bases de datos
