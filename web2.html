DEFINICIONES
Página web documento visible en la web mediante un programa llamado navegador, con hipervinculos o hiperenlaces a otras páginas web.

Hipervínculo elemento asociado a una acción de navegación y permite el acceso a otra página web o a otra aplicación web.

Sitio web  páginas con una estructura común que comparten un diseño y un mismo dominio.    

Página de entrada o principal que se suele llamar  index o inicio.
 
Marcos es una composición visual que divide la pantalla en zonas independientes.

Página web estática es una página realizada en lenguaje HTML con contenidos fijos
 
Página web dinámica es una página realizada en lenguajes basados en XML en la que el diseñador establece la estructura de la pagina y el contenido se introduce cuando esta es solicitada por el usuario.
